This is the beginning.

I built this site because I wanted a space that felt like mine — not a template, not a profile page on someone else's platform. Something with a bit of personality.

## The Stack

This site is built with [Astro](https://astro.build/) and styled with [Tailwind CSS](https://tailwindcss.com/). It's fast, static, and simple to maintain. The landing page has a custom particle system inspired by Touhou's bullet hell patterns — because if you're going to have a personal site, it might as well be a little extra.
